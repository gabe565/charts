## Custom configuration

### Authentication
Authentication can be enabled by setting the `KEYDB_PASSWORD` environment variable.

Example:
```yaml
controllers:
  keydb:
    containers:
      app:
        env:
          KEYDB_PASSWORD: changeme
```
